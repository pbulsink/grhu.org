About Us:

This is a chemistry calculation website, with a number of different calculations available or planned. So far available:

  • Elemental Analysis Calculator and Solvent Corrector
  • Table of the Isotopes

Planned, in development, or in testing:

  • Mass Spec fragment mass resolver
  • Formula Balancer
  • Mol - Mass Conversion

A sitemap is available here.

This is an open source project. Code can be seen at Github.
